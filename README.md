## ➤ Descripción General

**Hellsing-Attack** es una herramienta poderosa y versátil diseñada para pruebas de carga y auditorías de seguridad. Proporciona un conjunto completo de vectores de ataque para probar la resiliencia y el rendimiento de tu infraestructura. Ya seas un profesional de la seguridad, un administrador de sistemas o un desarrollador, Hellsing-Attack ofrece las herramientas que necesitas para garantizar que tus aplicaciones sean robustas y seguras.

## ➤ Características

- **Múltiples Vectores de Ataque**: Incluye Escaneo de Puertos, Inundación HTTP, Inundación TCP, Slowloris y un ataque DDoS simulado.
- **Interfaz Web y CLI**: Elige entre una interfaz de usuario interactiva basada en la web con Streamlit o una interfaz de línea de comandos tradicional.
- **Soporte Multilingüe**: Disponible en inglés y español.
- **Integración con Tor**: Anonimiza tus pruebas y evita las restricciones basadas en IP con el soporte incorporado de Tor y la rotación de identidad.
- **Configurable**: Configura fácilmente ajustes como el idioma, los hilos de trabajo y el uso de Tor.
- **Enfoque en el Uso Ético**: Diseñado para fines legítimos de prueba y auditoría.

## ➤ Instalación

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/fito422480/Hellsing-Attack.git
    cd Hellsing-Attack
    ```

2.  **Instala las dependencias:**
    ```bash
    pip install -r requirements.txt
    ```

## ➤ Uso

### Interfaz Web

Para lanzar la interfaz de usuario web fácil de usar, ejecuta:

```bash
streamlit run app.py
```

Esto abrirá una nueva pestaña en tu navegador con el panel de control de Hellsing-Attack.

### Interfaz de Línea de Comandos

Para usar la interfaz de línea de comandos, ejecuta:

```bash
python cli.py
```

Esto lanzará el menú interactivo de la CLI donde puedes seleccionar el ataque deseado y configurar sus parámetros.
